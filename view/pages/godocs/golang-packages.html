Understanding Go Packages

Learn how packages work in Go with simple examples

Packages are the fundamental building blocks of Go programs. Every Go file belongs to a package, and packages help organize code into reusable, maintainable units. Understanding packages is crucial for writing clean and modular Go code.

What are Packages?

A package in Go is a collection of related Go source files in the same directory. Packages provide:

  • Code organization: Group related functionality together
  • Reusability: Import and use code from other packages
  • Namespace: Prevent naming conflicts
  • Visibility control: Control what can be accessed from outside the package

Package Declaration

Every Go file must start with a package declaration. The package name must match the directory name (with some exceptions). 

package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

The package keyword declares which package this file belongs to.

The main Package

The main package is special. It defines an executable program. A program must have exactly one main() function in the main package. 

package main

import "fmt"

func main() {
    fmt.Println("This is an executable program")
    fmt.Println("You can run it with: go run main.go")
}

Note: Other packages (not named "main") are library packages that cannot be executed directly.

Importing Packages

Use the import statement to use code from other packages. Go provides many standard library packages, and you can also import your own packages.

Single Import

package main

import "fmt"

func main() {
    fmt.Println("Hello from fmt package")
}

Multiple Imports

package main

import (
    "fmt"
    "math"
    "time"
)

func main() {
    fmt.Println("Current time:", time.Now())
    fmt.Println("Square root of 16:", math.Sqrt(16))
}

Creating Your Own Packages

You can create your own packages by organizing files in directories. Each directory is a package.

Example: Create a calculator package

File: calculator/calculator.go

package calculator

// Add adds two numbers
func Add(a, b int) int {
    return a + b
}

// Subtract subtracts b from a
func Subtract(a, b int) int {
    return a - b
}

// Multiply multiplies two numbers
func Multiply(a, b int) int {
    return a * b
}

Using the package in main.go

File: main.go

package main

import (
    "fmt"
    "your-module/calculator"  // Import your package
)

func main() {
    sum := calculator.Add(5, 3)
    diff := calculator.Subtract(10, 4)
    product := calculator.Multiply(2, 7)
    
    fmt.Println("Sum:", sum)        // Output: Sum: 8
    fmt.Println("Difference:", diff)  // Output: Difference: 6
    fmt.Println("Product:", product)  // Output: Product: 14
}

Package Visibility (Exported vs Unexported)

In Go, visibility is controlled by the first letter of the identifier. Capitalized names are exported (public), lowercase names are unexported (private).

Example showing exported and unexported functions

package calculator

// Add is exported (can be used from other packages)
func Add(a, b int) int {
    return a + b
}

// subtract is unexported (only usable within this package)
func subtract(a, b int) int {
    return a - b
}

// Helper is an exported helper function
func Helper() {
    // Can call unexported functions from same package
    result := subtract(10, 5)
    // Use result...
}

Key point: From outside the package, you can only use Add() and Helper(). The subtract() function is private to the package.

Complete Working Example

Here's a complete example showing how to create and use a custom package:

Step 1: Create the package

File: greetings/greetings.go

package greetings

import "fmt"

// Greet returns a greeting message
func Greet(name string) string {
    return fmt.Sprintf("Hello, %s! Welcome to Go!", name)
}

// GreetMultiple greets multiple people
func GreetMultiple(names []string) {
    for _, name := range names {
        fmt.Println(Greet(name))
    }
}

Step 2: Use the package

File: main.go

package main

import (
    "fmt"
    "your-module/greetings"
)

func main() {
    // Use the Greet function
    message := greetings.Greet("Alice")
    fmt.Println(message)
    
    // Use the GreetMultiple function
    names := []string{"Bob", "Charlie", "Diana"}
    greetings.GreetMultiple(names)
}

Output:

Hello, Alice! Welcome to Go!
Hello, Bob! Welcome to Go!
Hello, Charlie! Welcome to Go!
Hello, Diana! Welcome to Go!

Key Takeaways

  • Every Go file belongs to a package
  • The main package creates an executable program
  • Capitalized names are exported (public), lowercase are unexported (private)
  • Packages help organize code into reusable, maintainable units
  • Import packages using the import keyword
